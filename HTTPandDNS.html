The Domain Name System
What is DNS?

-HTTP and DNS are closely related in the sense they contact servers
-DNS helps translate website domains into IP addresses.

How does DNS work?

Computers contact DNS servers by sending them a website such as "TheInternet.com" and asks them to search for the IP address. DNS then contacts different severs and retrieves the correct IP address, sending it back to the computer.

How does DNS help the internet scale?

DNS scales no matter how large the internet grows
